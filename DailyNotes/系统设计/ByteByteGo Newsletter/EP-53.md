### 五种SQL语句

- DDL: Data Definition Language;
- DML: Data Manipulation Language;
- DQL: Data Query Language;
- DCL: Data Control Language(GRANT, REVOKE);
- TCL: Transaction Control Language(COMMIT, ROLLBACK).
